A discord community around **httpc** is not yet active.
<br />
If you interested in joining such community, please leave a message in the [external:Github discussions](/discuss).
<br />

The community will focus on:
- httpc releases, architecture and behind the scene inner workings
- guides and tutorials with full project examples based on httpc
- support and clarification on existing httpc features
- feature requests
- general javascript/typescript API design

<br />
If you're looking for something more or different, please specify it in message at [external:Github discussions](/discuss).
